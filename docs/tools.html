Tools & Validation

Tools for validating and testing version detection patterns

Our tools help ensure the quality and reliability of patterns in the database. All tools are open-source and available in the tools directory of our repository.

Pattern Validation Script

The pattern validation script validates JSON files against the expected schema and tests the regex patterns with provided test cases.

Features
  • JSON schema validation
  • Regex compilation testing
  • Test case execution
  • Version extraction validation
Requirements

Python 3.6+ with jsonschema package:

pip install jsonschema
Usage
python tools/validate-pattern.py patterns/web/apache.json
Multiple Files
python tools/validate-pattern.py patterns/web/apache.json patterns/networking/openssh.json
Example Output
