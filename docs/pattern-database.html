Pattern Database

The Version Detection Pattern Database contains regex patterns used to identify software versions from various sources like HTTP headers, file contents, network responses, and more. These patterns are used by security tools to detect vulnerable software versions during assessments.

Using the Patterns

All patterns are stored as JSON files in the patterns directory of our repository. Each pattern file contains:

  • Name and category information
  • Regex pattern for version detection
  • Vendor and product details
  • Priority and confidence scores
  • Metadata with description and tags
  • Test cases for validation

You can browse the patterns directory to see all available patterns, or check our pattern summary for an overview of the database.
