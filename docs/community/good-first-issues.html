Good First Issues

Welcome! If you're new to contributing to open-source or to this project specifically, these issues are a great place to start. They're well-defined, relatively self-contained, and don't require deep knowledge of the entire codebase.

Advisory Creation

Easy

Create Advisory for Recent CVE

Research a recently published CVE and create an advisory entry for it following our template.

Skills needed: Research skills, attention to detail

Expected outcome: A new advisory file in advisories/cve/ with comprehensive references

Easy

Create Advisory for Malware Family

Research a malware family and create an advisory entry with detection and mitigation information.

Skills needed: Research skills, attention to detail

Expected outcome: A new advisory file in advisories/malware/

Medium

Create Advisory for APT Campaign

Research an APT group's recent campaign and create a comprehensive advisory with TTPs and indicators.

Skills needed: Advanced research skills, understanding of threat intelligence

Expected outcome: A new advisory file in advisories/apt/

Advisory Improvement

Easy

Add References to Existing Advisory

Find additional authoritative sources for an existing advisory and add them to the references section.

Skills needed: Research skills, attention to detail

Expected outcome: Updated references in an existing advisory file

Easy

Improve Recommendations for Existing Advisory

Review existing advisories and enhance their recommendations with more actionable guidance.

Skills needed: Cybersecurity knowledge, clear writing skills

Expected outcome: Enhanced recommendations in an existing advisory

Documentation

Easy

Improve Advisory Template Documentation

Enhance the advisory template documentation with better examples and explanations.

Skills needed: Writing skills, understanding of the project

Expected outcome: Improved advisories/TEMPLATE.md file

Easy

Create Examples for Different Categories

Create example advisories for categories that don't have good examples yet.

Skills needed: Research skills, attention to detail

Expected outcome: New example advisory files in underrepresented categories

Tooling

Medium

Enhance Validation Tool

Add new validation features to the advisory validation tool, such as checking for common data pitfalls.

Skills needed: Python programming, understanding of data validation

Expected outcome: Enhanced tools/validate-advisory.py

How to Get Started

  1. Read our Beginner's Guide if you haven't already
  2. Choose an issue that interests you and matches your skill level
  3. Fork the repository and create a new branch for your work
  4. Comment on the issue to let others know you're working on it
  5. Make your changes following our contribution guidelines
  6. Submit a pull request when you're ready for review
